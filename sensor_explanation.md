# LDR Sensor Working

An LDR (Light Dependent Resistor) is a sensor whose resistance changes according to the amount of light falling on it.

When light intensity increases, resistance decreases.
When light intensity decreases, resistance increases.

In this project the LDR is connected to the Arduino analog pin A0.

The Arduino continuously reads the light value.

If the light intensity falls below a threshold, the Arduino turns ON the LED.

This system simulates an automatic lighting system used in smart street lights.
